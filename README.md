# 📝 Flask To-Do List Application

A simple and efficient **To-Do List Web Application** built using **Flask** that helps users manage their daily tasks. Users can add, update, mark tasks as completed, and delete tasks through an easy-to-use web interface.

---

## 🚀 Features

- ✅ Add new tasks
- ✏️ Edit existing tasks
- ✔️ Mark tasks as completed
- 🗑️ Delete tasks
- 📋 View all tasks

---

## 🛠️ Tech Stack

- **Backend:** Flask (Python)
- **Frontend:** HTML
